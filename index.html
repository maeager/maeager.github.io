Imaging

SIMPLE (Restricted) Single-particle electron microscope image processing engine (simplecryoem.com).

SuperRes StriaOfGenari Super-resolution MR imaging of visual cortex using single coil and complex image processing. This project was devised by the Monash Bionic Vision group and Monash Biomedical Imaging facility to assertain the viability of non-sighted subjects' V1 area, particularly the Stria of Gennari, for possible cortical implantation of a bionic vision device.

Xglom Multi-modal kidney imaging application (MKIA) -- a quantification of ferritin-labelled glomeruli in high-field MR, synchrotron CT and microscopic images.

Agilent2Dicom Image processing application for Monash Biomedical Imaging's Agilent 9.4T MR scanner.

Previs (Restricted) Web volume and mesh application for Monash Immersive Visualisation Platform.

PhD thesis

thesis-eagerm My manuscript was written in org-mode markup, a subpackage of Emacs, then exported to latex. This enabled in-text code replacement, an essential tool for ensuring reproducible research results.

PhD Supervisors: David Grayden (@dgrayden), Anthony Burkitt and Hamish Meffin.

Neuro

cnstellate A biophysically-realistic neural microcircuit of the cochlear nucleus
Neural models were written in NEURON based on the Rothman and Manis CN stellate models, with auditory inputs generated in C++ models from Carney Lab.

GAneuron Simultaneous parameter optmisation of neural model parameters using parallel genetic algorithm.

SpikeSim A large-scale recurrent neural network simulator
